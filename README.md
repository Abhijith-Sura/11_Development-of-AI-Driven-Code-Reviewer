# Iris Flower Classification Project

**Infosys Springboard AIML Virtual Internship - Individual Task**

## What This Project Does

This project classifies Iris flowers into 3 types (Setosa, Versicolor, Virginica) using machine learning.

## Three Components

1. **Model Research** - Compared 3 different ML models
2. **Python Task** - Built ML model with 100% accuracy
3. **SQL Task** - Stored data in MySQL database

## Project Structure

iris_classification/
├── data/ # Dataset files
├── models/ # Trained ML models
├── src/ # Python code
├── sql/ # Database setup
├── model_research.md # Research document
├── requirements.txt # Required libraries
└── README.md # This file



## How to Run

### 1. Install libraries
pip install -r requirements.txt



### 2. Train the model
python src/train_model.py



### 3. Test predictions
python src/predict.py



### 4. Test database
python src/database.py
